This game is being created as my final project for my computer science class.
This is a roguelike deckbuilder in the style of games like Balatro based off of a bingo board.
Game is in very early development, but I am adding new stuff often.
The game is currently only playable in an IDE terminal, but after base mechanics are finished being added, I will use PyGame to create a user interface.
If you come across this game, please enjoy!
